## 🎯 My Vision

I'm building a **unified operating system for traditional manufacturing and trading businesses** — one connected platform where **accounting, inventory, sales, dispatch, and shop-floor operations** all speak the same language, in real time.

Most small and mid-sized businesses still juggle a legacy ERP for accounts, paper for the warehouse, WhatsApp for orders, and spreadsheets for everything in between. I want to replace that chaos with a single, reliable system:

- 🖥️ **Desktop** for accountants
- 📱 **Android** for operations and shop-floor staff
- 🌐 **Web** for admins and owners
- ☁️ **Cloud backend** that plays nicely with existing ERP systems

Long term, my mission is to become the **software backbone that traditional businesses have always needed but never had** — bridging decades-old tools with modern, mobile-first workflows, without forcing anyone to throw away what already works.

## 🚀 What I'm Building

| Layer | What it does |
|---|---|
| 🧠 **Core backend** | Go services for inventory, vouchers, ledgers, dispatch, tax config, and images |
| 🛠️ **Admin backend** | Go service for company, user, device, and backup management |
| 🌉 **ERP middleware** | Bridges mobile/desktop apps with legacy accounting systems |
| 📱 **Operations app** | Android-first Flutter app for sales, production, packing, and inventory |
| 🖥️ **Accountant desktop** | Windows Flutter app connecting accountants directly to the ERP |
| 🌐 **Admin web panel** | Flutter Web admin for companies, users, devices, and configuration |
| 📦 **Dispatch module** | QR-driven warehouse packing stations with role-based workflows |
